# Python Projects Repository

## Introduction
This repository contains a collection of Python projects built to practice and demonstrate core programming concepts such as data types, data structures, and object-oriented programming (OOP).  

These projects are designed to simulate real-world systems and improve problem-solving, code organization, and logical thinking skills.

---

## Projects Overview

### 1. Student Management System
**Description:**  
A system used to manage student records including adding students, storing their information, and retrieving data using structured data handling.

**Key Concepts:**
- Dictionaries
- Lists
- OOP (Classes and Objects)
- Encapsulation


### 2. Expense Tracker System
**Description:**  
A simple application to track daily expenses, categorize them, and calculate total spending.

**Key Concepts:**
- Lists
- Dictionaries
- OOP
- Data aggregation


### 3. Library Management System
**Description:**  
A system that manages books in a library, allowing borrowing and returning of books while tracking availability.

**Key Concepts:**
- OOP
- Encapsulation
- Lists
- Condition handling

### 4. Inventory Management System
**Description:**  
A system used to manage product inventory, update stock quantities, and track available products.

**Key Concepts:**
- Dictionaries
- OOP
- Data handling
- State management


## Concepts Covered
- Python Data Types (int, float, string, boolean)
- Data Structures (Lists, Tuples, Dictionaries, Sets)
- Object-Oriented Programming:
  - Encapsulation
  - Inheritance
  - Polymorphism
  - Abstraction
